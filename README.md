# User Authentication System

A full-stack authentication system with JWT-based sessions, email verification, and password reset functionality.

## Features

- Email/Password authentication
- JWT session management (access + refresh tokens)
- Email verification
- Password reset flow
- Session management (view/revoke active sessions)
- Rate limiting and security best practices

## Planned Improvements

- OAuth integration (Google, GitHub, etc.)
- Enhanced UI/UX design

## Tech Stack

**Frontend:** React, TypeScript, TanStack Query, React Router, Vite

**Backend:** Node.js, Express, TypeScript, MongoDB, JWT, Bcrypt

**Email:** Resend
